====== Aluminium case ======

{{./pasted_image.png}}
The **aluminium case** is an intermediate crafting ingredient used to make  {{~/.minetest/games/loria/mods/loria/textures/loria_battery.png}} [[Battery]].

====== Obtaining ======

===== Furnace crafting =====
1x  {{~/.minetest/games/loria/mods/loria/textures/loria_aluminium_ingot.png}} [[Aluminium ingot]] → 1x  {{~/.minetest/games/loria/mods/loria/textures/loria_aluminium_case.png}} **Aluminium case**
This recipe takes 3 units of time.

====== Usage ======

===== Refiner crafting =====
1x  {{~/.minetest/games/loria/mods/loria/textures/loria_mercury_oxide.png}} [[Mercury (II) oxide (HgO)]] + {{~/.minetest/games/loria/mods/loria/textures/bucket_potassium_hydroxide.png}} [[Potassium hydroxide bucket]] + 1x  {{~/.minetest/games/loria/mods/loria/textures/loria_zinc_ingot.png}} [[Zinc ingot]] + 1x  {{~/.minetest/games/loria/mods/loria/textures/loria_aluminium_case.png}} **Aluminium case** → 1x  {{~/.minetest/games/loria/mods/loria/textures/loria_battery.png}} [[Battery]] + 1x  {{~/.minetest/games/loria/mods/loria/textures/bucket.png}} [[Empty bucket]]
This recipe takes 2 units of time.

1x  {{~/.minetest/games/loria/mods/loria/textures/loria_red_mercury_oxide.png}} [[Mercury (II) oxide (HgO, red)]] + {{~/.minetest/games/loria/mods/loria/textures/bucket_potassium_hydroxide.png}} [[Potassium hydroxide bucket]] + 1x  {{~/.minetest/games/loria/mods/loria/textures/loria_zinc_ingot.png}} [[Zinc ingot]] + 1x  {{~/.minetest/games/loria/mods/loria/textures/loria_aluminium_case.png}} **Aluminium case** → 1x  {{~/.minetest/games/loria/mods/loria/textures/loria_battery.png}} [[Battery]] + 1x  {{~/.minetest/games/loria/mods/loria/textures/bucket.png}} [[Empty bucket]]
This recipe takes 2 units of time.
